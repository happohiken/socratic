# socratic
Un lector de PDFs interactivo que utiliza un LLM para convertir cualquier documento en una experiencia de estudio basada en el diálogo.

Socratic es una aplicación de estudio interactivo que transforma cualquier documento en una conversación. Mientras lees un PDF, puedes interrumpir en cualquier momento para hacer preguntas, pedir aclaraciones, profundizar en conceptos o solicitar ejemplos, y continuar la lectura exactamente donde la dejaste. Utiliza un modelo de lenguaje (LLM) como tutor inteligente para convertir el estudio pasivo en un aprendizaje activo.
